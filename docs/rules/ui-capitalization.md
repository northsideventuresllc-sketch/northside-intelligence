# UI Capitalization Rules

**Scope:** Northside Intelligence portal and Sector 3 tools.

## Buttons and CTAs

Use **Title Case** for all button text, links styled as buttons, and primary actions.

| Correct | Avoid |
|---------|-------|
| Get Started | Get started |
| Sign In | Sign in |
| Create Account | Create account |
| Open Tool | Open Tool → |
| Explore Tools | explore tools |

For smaller or compact controls (tiny pills, icon-only adjacent labels), **ALL CAPS** is acceptable (e.g. category filter pills).

## Headings and titles

Section headings, card titles, and bold UI labels use Title Case unless the user explicitly requests sentence case.

## Body copy

Paragraphs, descriptions, tooltips, and long-form legal or help text use normal sentence capitalization.
